Selecting

All

Reference

Mode:

All modes

Menu:

Select ‣ All

Shortcut:

A

Selects all items.

None

Reference

Mode:

All modes

Menu:

Select ‣ None

Shortcut:

Alt-A

Resets the selection to nothing.

Invert

Reference

Mode:

All modes

Menu:

Select ‣ Inverse

Shortcut:

Ctrl-I

Selects non-selected items and deselects existing selection.

Reference

Mode:

All modes

Menu:
